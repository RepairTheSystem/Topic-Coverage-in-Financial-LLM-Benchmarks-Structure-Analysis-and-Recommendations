# Topic Coverage in Financial LLM Benchmarks: Structure Analysis and Recommendations

This repository contains supplementary materials for the paper **“Topic Coverage in Financial LLM Benchmarks: Structure Analysis and Recommendations”**

## What is in this repository

- extended descriptions of the benchmarks used in the study
- manual topic-validation criteria
- the full topic/subtopic appendix
- supplementary figures and materials related to the experiments

## Summary

The paper analyzes topic coverage in existing benchmarks for financial large language models (FinLLMs).

- **9 datasets**
- **44,507 tasks**
- **43,978 records after preprocessing**
- **16 high-level topics**
- **250 subtopics**
- **27,379 records in the final taxonomy**

Topic extraction was performed with **BERTopic**, followed by **manual validation and refinement**.

## Manual operations

Each cluster was assigned one of four actions:

- **Keep as is**
- **Rename**
- **Merge**
- **Discard**

These criteria are described in the supplementary materials.

## Purpose

The repository is intended to improve transparency and reproducibility by providing the materials referenced in the paper but omitted from the main text due to space limitations.
